# 1부: 공통 화면 (비로그인 사용자)

---

## 1.1 홈페이지 (`/`)

홈페이지는 사이트에 접속했을 때 가장 먼저 보이는 화면입니다.

### 화면 구성
| 영역 | 설명 |
|------|------|
| 히어로 섹션 | 서비스 소개 문구와 주요 CTA(Call to Action) 버튼 |
| 일자리 미리보기 | 최근 등록된 일자리 목록 일부 표시 |
| 통계 | 등록 회원 수, 고객사 수 등 서비스 규모 표시 |
| 서비스 소개 | Humend HR 서비스의 주요 특징 설명 |

### 버튼 및 기능
| 버튼/링크 | 위치 | 동작 |
|-----------|------|------|
| **일자리 보기** | 히어로 섹션 | `/jobs` 일자리 목록 페이지로 이동 |
| **회원가입** | 상단 네비게이션 | `/signup` 회원가입 페이지로 이동 |
| **로그인** | 상단 네비게이션 | `/login` 로그인 페이지로 이동 |
| **마이페이지** | 상단 네비게이션 (로그인 시) | `/my` 마이페이지로 이동 |

---

## 1.2 일자리 목록 (`/jobs`)

현재 모집 중인 모든 일자리를 카드 형태로 보여주는 페이지입니다.

### 화면 구성
| 영역 | 설명 |
|------|------|
| 날짜 필터 | 시작일~종료일 범위로 일자리 검색 |
| 일자리 카드 그리드 | 고객사별 일자리 카드 (3열/2열/1열 반응형) |
| 빈 상태 메시지 | 조건에 맞는 일자리가 없을 때 표시 |

### 버튼 및 기능
| 버튼/링크 | 위치 | 동작 |
|-----------|------|------|
| **시작일 선택** | 필터 영역 | 날짜 선택기로 검색 시작일 설정 |
| **종료일 선택** | 필터 영역 | 날짜 선택기로 검색 종료일 설정 |
| **초기화** | 필터 영역 | 날짜 필터 해제, 전체 일자리 표시 |
| **지원하기** | 각 일자리 날짜 슬롯 | 해당 날짜/시간의 일자리에 지원 |

### 일자리 카드 정보
각 카드에 표시되는 정보:
- 고객사 이미지 또는 아이콘
- 고객사명 (클릭 시 상세 페이지 이동)
- 근무지 위치
- 시급
- 모집 건수 배지
- 최근 3개 날짜의 근무시간 및 지원 버튼

### 지원하기 버튼 동작
1. 버튼 클릭 → 확인 다이얼로그 표시
2. **프로필 미완성 시**: 에러 모달 → "프로필을 먼저 완성해주세요" 메시지 → 프로필 페이지로 안내
3. **프로필 완성 시**: 지원 확인 → 성공 다이얼로그 표시
4. **로그인 필요 시**: 로그인 페이지로 리다이렉트

## 1.3 일자리 상세 (`/jobs/[id]`)

특정 일자리의 상세 정보를 보여주는 페이지입니다.

### 화면 구성
| 영역 | 설명 |
|------|------|
| 이미지 갤러리 | 고객사/현장 사진 1~3장 표시 |
| 기본 정보 | 고객사명, 위치, 시급 (크게 표시) |
| 정보 카드 (4열) | 모집인원, 업무형태, 성별 조건, 업종 |
| 근무 일정 표 | 날짜별 시간/인원/지원 버튼 |
| 상세 정보 | 업무 설명, 복장, 근무 유의사항, 담당자 연락처 |
| 지도 | 카카오맵으로 근무지 위치 표시 |

### 버튼 및 기능
| 버튼/링크 | 위치 | 동작 |
|-----------|------|------|
| **지원하기** | 각 근무 슬롯 옆 | 해당 날짜/시간에 지원 (확인 다이얼로그 포함) |
| **뒤로가기** | 상단 | 일자리 목록으로 돌아가기 |
| **담당자 전화** | 하단 연락처 | 전화번호 표시 |

# 2부: 회원 메뉴

> 로그인한 회원만 접근 가능한 페이지들입니다.
> 모든 페이지는 `/my` 경로 하위에 있으며, 비로그인 시 자동으로 로그인 페이지로 이동합니다.

---

## 2.1 마이페이지 (대시보드) (`/my`)

로그인 후 가장 먼저 보이는 회원 전용 메인 페이지입니다.

### 화면 구성

#### 프로필 영역 (상단)
| 요소 | 설명 |
|------|------|
| 프로필 사진 | 등록된 사진 표시, 미등록 시 이니셜 아이콘 |
| 이름 | 회원 이름 표시 |
| 환영 메시지 | "님, 안녕하세요" |
| 프로필 완성도 | 프로필 입력 비율을 퍼센트(%)로 표시 |

#### 상태 카드 (3개)
| 카드 | 표시 내용 | 색상 |
|------|-----------|------|
| **대기 중 지원** | 승인 대기 중인 지원 건수 | 앰버(노란색) |
| **승인된 지원** | 승인 완료된 지원 건수 | 에메랄드(녹색) |
| **이번 달 급여** | 당월 수령 급여 총액 | 슬레이트(회색) |

#### 빠른 링크 (4개)
| 링크 | 아이콘 | 이동 경로 | 설명 |
|------|--------|-----------|------|
| **프로필 관리** | 사람 아이콘 | `/my/resume` | 회원정보 등록/수정 |
| **근무신청 조회** | 클립보드 아이콘 | `/my/applications` | 내 지원 현황 확인 |
| **근무내역** | 시계 아이콘 | `/my/history` | 월별 근무내역 조회 |
| **급여신청** | 지갑 아이콘 | `/my/salary` | 계약체결 및 급여신청 |

#### 예정 근무 영역
승인된 지원 중 앞으로 다가오는 근무 일정을 D-day로 표시합니다.
- **TODAY**: 녹색 배지 (오늘 근무)
- **D-3 이내**: 노란색 배지 (임박한 근무)
- **그 외**: 회색 배지

#### 프로필 미완성 안내
프로필이 완성되지 않은 경우 하단에 "이력서를 등록해주세요" CTA 카드가 표시됩니다.

### 버튼 및 기능
| 버튼/배지 | 위치 | 동작 |
|-----------|------|------|
| **설정** 배지 | 프로필 영역 | 비밀번호 변경 기능 |
| **회원탈퇴** 배지 | 프로필 영역 | 계정 삭제 확인 다이얼로그 → 탈퇴 처리 |
| **빠른 링크 카드** | 중앙 그리드 | 각 해당 페이지로 이동 |
| **이력서 등록** CTA | 하단 | `/my/resume`로 이동 |
| **전체보기** | 예정 근무 영역 | `/my/applications`로 이동 |

## 2.2 프로필 관리 (이력서) (`/my/resume`)

회원 정보를 등록하고 수정하는 핵심 페이지입니다. 일자리에 지원하려면 프로필을 완성해야 합니다.

### 화면 구성 (섹션별)

#### 1) 프로필 사진
| 요소 | 설명 |
|------|------|
| 프로필 이미지 | 원형으로 표시, 미등록 시 이니셜 표시 |
| 카메라 버튼 | 사진 업로드/변경 |

**사진 업로드 동작:**
1. 카메라 아이콘 클릭
2. 파일 선택 대화상자 표시
3. 이미지 선택 → 자동 압축 (최대 0.5MB, 1024px)
4. 업로드 후 즉시 반영

#### 2) 기본 정보 섹션 (파란색 헤더)
| 필드 | 타입 | 설명 | 편집 가능 |
|------|------|------|-----------|
| **이름** | 텍스트 | 가입 시 입력한 이름 | X (읽기 전용) |
| **전화번호** | 텍스트 | 가입 시 입력한 번호 | X (읽기 전용) |
| **생년월일** | 날짜 선택 | 생년월일 입력 | O |
| **성별** | 라디오 버튼 | 남성/여성 선택 | O |
| **이메일** | 이메일 입력 | 이메일 주소 | O |
| **지역** | 텍스트 입력 | 거주 지역 | O |
| **신장** | 숫자 입력 | 키 (cm) | O |

#### 3) 본인인증 섹션 (녹색 헤더)
| 필드 | 타입 | 설명 |
|------|------|------|
| **주민번호 앞자리** | 숫자 입력 | 6자리 |
| **주민번호 뒷자리** | 마스크 입력 | 7자리 (보안 처리) |
| **NICE 인증** 버튼 | 버튼 | 본인인증 실행 |
| **인증 상태** | 배지 | 인증 완료/미인증 표시 |

**인증 동작:** NICE 인증 버튼 클릭 → 본인인증 프로세스 진행 → 완료 시 "인증 완료" 배지 표시

#### 4) 경력 사항 섹션 (보라색 헤더)
| 필드 | 타입 | 설명 |
|------|------|------|
| **경력 여부** | 라디오 버튼 | 있음/없음 선택 |
| **경력 상세** | 텍스트 영역 | "있음" 선택 시에만 표시, 경력 내용 자유 입력 |

#### 5) 계좌 정보 섹션 (주황색 헤더)
| 필드 | 타입 | 설명 |
|------|------|------|
| **은행명** | 드롭다운 | 21개 은행 중 선택 |
| **예금주** | 텍스트 입력 | 계좌 소유자 이름 |
| **계좌번호** | 숫자 입력 | 자동 하이픈 포맷 적용 |

**지원 은행 목록:** KB국민, 신한, 우리, 하나, NH농협, IBK기업, SC제일, 씨티, 카카오뱅크, 케이뱅크, 토스뱅크, 새마을금고, 신협, 우체국, 수협, 대구, 부산, 광주, 전북, 경남, 제주

#### 6) 개인정보 동의 섹션 (회색 헤더)
| 요소 | 설명 |
|------|------|
| 약관 내용 | 스크롤 가능한 개인정보처리방침 전문 |
| **동의 체크박스** | 개인정보 수집/이용에 동의 (필수) |

### 버튼 및 기능
| 버튼 | 위치 | 동작 |
|------|------|------|
| **카메라 아이콘** | 프로필 사진 위 | 사진 파일 업로드 대화상자 |
| **남성/여성 토글** | 기본 정보 | 성별 선택 |
| **있음/없음 토글** | 경력 섹션 | 경력 여부 선택, "있음" 시 텍스트 영역 표시 |
| **은행 선택 드롭다운** | 계좌 정보 | 은행 목록에서 선택 |
| **NICE 인증** | 본인인증 섹션 | 본인인증 프로세스 실행 |
| **저장하기** (제출) | 페이지 하단 | 전체 프로필 정보 저장 |

### 유효성 검증 모달
저장 시 필수 항목이 비어있으면 검증 모달이 표시됩니다:
- 비어있는 필드 목록이 표시됨
- 각 항목 클릭 시 해당 필드로 자동 스크롤
- 닫기 버튼으로 모달 닫기

### 저장 성공 모달
저장 완료 시 성공 메시지 모달이 표시되고, 마이페이지로 리다이렉트됩니다.

## 2.3 근무신청 조회 (`/my/applications`)

| **승인** | 에메랄드(녹색) 배경 | 승인된 지원 건수 |
| **거절** | 레드(빨간색) 배경 | 거절된 지원 건수 |

#### 탭 메뉴
| 탭 | 표시 내용 |
|------|-----------|
| **전체** (건수) | 모든 지원 내역 |
| **대기** (건수) | 승인 대기 중인 내역만 |
| **승인** (건수) | 승인된 내역만 |
| **거절** (건수) | 거절된 내역만 |

#### 지원 내역 리스트

| 정보 | 설명 |

| 근무일 + 시간 | 근무 예정 날짜와 시간대 |

| 상태 배지 | 대기중(노란) / 승인(녹색) / 거절(빨간) |

### 버튼 및 기능
| 버튼 | 동작 |
|------|------|
| **탭 전환** | 해당 상태의 지원 내역만 필터링하여 표시 |

> 이 페이지에서는 조회만 가능하며, 지원 취소 등의 액션 버튼은 없습니다.

## 2.4 급여신청 (`/my/salary`)

| 근무시간 | 시작~종료 시간 |
| 상태 배지 | 대기(노란) / 확정(녹색) / 지급완료(파란) |
| **계약 체결** 버튼 | 계약 모달 열기 → 서명 후 계약 체결 |

#### 서명 완료 & 지급 대기 기록

| 정보 | 설명 |
|------|------|
| 상태 | "대기" 표시 |
| **계약서 보기** 버튼 | 서명된 계약서 PDF 보기 모달 |

#### 2) 계약서 보기 모달

#### 3) 급여명세서 모달

|------|------|
| 총 급여 | 세전 급여 금액 |
| 공제 내역 | 세금 등 공제 항목 |
| 실수령액 | 세후 실제 수령 금액 |
| 지급일/상태 | 급여 지급 일자 및 상태 |
| **닫기** 버튼 | 모달 닫기 |

#### 4) 직접 급여신청 모달
| 요소 | 설명 |
|------|------|
| **고객사 선택** 드롭다운 | 급여를 신청할 고객사 선택 |

| **신청** 버튼 | 급여 신청 제출 |
| **취소** 버튼 | 모달 닫기 |

### 버튼 요약

|------|------|------|

## 3.1 관리자 대시보드 (`/admin`)

### 화면 구성

#### 통계 카드 (5개, 접기/펴기 가능)
| 카드 | 아이콘 | 색상 | 표시 내용 |
|------|--------|------|-----------|
| **확정 근무** | 달력 체크 | 파랑-인디고 | 확정된 근무 기록 수 |

| **등록 회원** | 사람들 | 보라-퍼플 | 전체 등록 회원 수 |
| **제휴 고객사** | 건물 | 에메랄드-틸 | 등록된 고객사 수 |

| 차트 | 유형 | 설명 |
|------|------|------|

| **월별 급여** | 바 차트 | 최근 4개월 급여 추이 |

#### 미처리 지원 카드
| 요소 | 설명 |
|------|------|
| 지원 목록 | 최근 5건의 대기 중 지원 표시 |
| 회원명 | 지원한 회원 이름 |
| 고객사 | 지원한 고객사 |
| 지원일 | 지원 날짜 |
| **전체보기** 링크 | `/admin/applications`로 이동 |
| **+N건 더보기** | 5건 초과 시 표시 |

#### 최근 활동 타임라인
| 요소 | 설명 |
|------|------|
| 상태 점 | 녹색(승인)/빨간색(거절)/노란색(대기) |
| 회원명 | 관련 회원 이름 |
| 고객사 | 관련 고객사 |
| 행동 | 지원/승인/거절 등 |
| 날짜 | 활동 일시 |
| 상태 배지 | 현재 상태 |

### 버튼 및 기능
| 버튼/링크 | 동작 |
|-----------|------|
| **카드 접기/펴기** | 통계 카드 영역 토글 |
| **차트 접기/펴기** | 차트 영역 토글 |

## A. 상태 배지 색상 정리

| 상태 | 색상 | 사용 위치 |
|------|------|-----------|
| 대기 / 대기중 | 앰버 (노란색) | 지원 상태, 급여 상태 |
| 승인 | 에메랄드 (녹색) | 지원 승인 |
| 거절 | 레드 (빨간색) | 지원 거절 |
| 확정 | 에메랄드 (녹색) | 근무 확정 |
| 지급완료 | 블루 (파란색) | 급여 지급 완료 |
| 완료 | 에메랄드 (녹색) | 계약 완료 |
| 모집중 (open) | 녹색 | 일자리 모집 |
| 마감 (closed) | 회색 | 일자리 마감 |

## B. 주요 업무 흐름

### 회원 업무 흐름
```
1. 회원가입 → 로그인
2. 프로필 등록 (필수: 기본정보 + 계좌정보 + 개인정보 동의)
3. 일자리 탐색 → 지원하기
4. 지원 상태 확인 (대기 → 승인/거절)
5. 승인 시: 근무 후 급여신청 페이지에서 계약 체결 (서명)
6. 서명 완료 후 급여 지급 대기
7. 급여명세서 확인
```

### 관리자 업무 흐름
```
1. 관리자 로그인
2. 고객사 등록/관리
3. 일자리(근무 슬롯) 등록
4. 회원 지원 접수 → 승인/거절 처리
5. 근무 완료 후 급여 승인
6. 급여 지급 처리 및 확인
7. 계약서/지급내역 관리
```

## C. 반응형 디자인

Humend HR은 데스크탑, 태블릿, 모바일에서 모두 사용 가능합니다:
- **데스크탑**: 모든 테이블 열 표시, 넓은 그리드 레이아웃
- **태블릿**: 일부 열 숨김, 2열 그리드
- **모바일**: 핵심 정보만 표시, 1열 레이아웃, 터치 최적화
