05 — Layout Patterns

레이아웃 패턴

HumendHR 프로필 페이지의 주요 레이아웃 구조와 설계 원칙입니다.

1Sidebar + Content

좌측 고정 사이드바(~180px) + 우측 스크롤 메인 콘텐츠. 브랜드, 네비게이션, 하단 로그아웃.

2Main + Right Panel

메인(~65%) + 우측 패널(~35%) 2컬럼. Recent Activity, Holidays가 우측 배치.

3Profile Banner

블루 그래디언트 배너 + overlap 아바타. 프로필 완성도 링과 액션 버튼 우측 배치.

4Card Sections

정보를 카드 단위로 그룹핑. 카드 내 label-value 그리드(2열) 패턴 반복.

5Tab Navigation

배너 하단 About Me / Employment / Documents / Team 탭. 언더라인 인디케이터(2px).

6Bottom Stats Bar

하단에 Leave Balance, Years 등 핵심 통계를 블루 배경 카드로 나란히 배치.

06 — Design Principles

디자인 원칙

HumendHR 디자인 시스템의 핵심 설계 원칙입니다.

Clarity First
HR 정보는 명확하게. Label-Value 패턴으로 데이터를 구조화하고, 시각적 계층을 뚜렷이.
Consistent Cards
모든 정보 그룹은 동일한 카드 패턴. 좌측 블루 바 타이틀, 우측 Update 링크 일관 적용.
Trust & Calm
블루 프라이머리로 신뢰감, 라이트 테마로 전문적이고 차분한 인상. 과도한 장식 지양.
Subtle Feedback
hover 시 미세한 lift+shadow. focus 시 블루 ring. 120-200ms 전환으로 자연스러운 인터랙션.
