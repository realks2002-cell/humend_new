Visual User Manual

Humend HR
비주얼 사용자 매뉴얼

인력파견 플랫폼 Humend HR의 전체 화면과 기능을 스크린샷과 함께 안내합니다. 회원(파견 근로자)과 관리자(파견업체) 두 역할의 모든 메뉴를 다룹니다.

데스크탑 + 모바일 반응형
회원 + 관리자 2역할
32개 실제 화면 캡처
01 — Public Pages

공통 화면

로그인 없이 누구나 접근할 수 있는 페이지입니다. 홈페이지, 일자리 목록/상세, 로그인, 회원가입, 소개, 관리자 로그인을 포함합니다.

1.1
홈페이지
/

사이트에 접속했을 때 가장 먼저 보이는 화면입니다. 히어로 섹션, 최근 알바공고, 서비스 통계, 소개 영역으로 구성됩니다.

humendhr.com
버튼/링크위치동작
일자리 보기히어로 섹션/jobs 일자리 목록으로 이동
회원가입상단 네비게이션/signup 회원가입 페이지로 이동
로그인상단 네비게이션/login 로그인 페이지로 이동
마이페이지상단 네비게이션 (로그인 시)/my 마이페이지로 이동
1.2
일자리 목록
/jobs

현재 모집 중인 모든 일자리를 카드 형태로 보여주는 페이지입니다. 날짜 필터로 원하는 기간의 일자리를 검색할 수 있습니다.

humendhr.com/jobs
기능설명
날짜 필터시작일~종료일 범위로 일자리 검색
일자리 카드고객사 이미지, 위치, 시급, 모집건수, 날짜별 지원 버튼
지원하기프로필 완성 시 지원 확인 다이얼로그 → 미완성 시 프로필 페이지 안내
1.3
일자리 상세
/jobs/[id]

특정 일자리의 상세 정보를 보여주는 페이지입니다. 이미지 갤러리, 기본 정보, 모집 일정, 상세 설명, 지도 등을 포함합니다.

humendhr.com/jobs/[id]
영역내용
이미지 갤러리고객사/현장 사진 1~3장 표시
정보 카드모집인원, 업무형태, 성별 조건, 업종
모집 일정날짜별 시간/인원/지원 버튼
복장/가이드준비물 SVG 가이드, 주의사항
지도카카오맵으로 근무지 위치 표시
1.4
회원 로그인
/login

전화번호와 비밀번호로 로그인합니다. 로그인 성공 시 마이페이지(/my)로 이동합니다.

humendhr.com/login
1.5
회원가입
/signup

이름, 전화번호, 비밀번호를 입력하여 간편하게 가입합니다. 가입 완료 후 로그인 페이지로 이동합니다.

humendhr.com/signup
1.6
사업소개
/about

Humend HR 서비스의 프로세스(공고 등록 → 인력 매칭 → 근무 확인 → 급여 정산)와 핵심 가치를 소개합니다.

humendhr.com/about
1.7
관리자 로그인
/admin/login

관리자 전용 로그인 페이지입니다. 관리자 ID와 비밀번호로 인증 후 관리자 대시보드로 이동합니다. 좌측에 관리자 사이드바 네비게이션이 표시됩니다.

humendhr.com/admin/login
02 — Member Pages

회원 메뉴

로그인한 회원만 접근 가능한 페이지입니다. 비로그인 시 자동으로 로그인 페이지로 이동합니다.

회원가입
→
프로필 등록
→
일자리 지원
→
승인 대기
→
근무
→
계약 서명
→
급여 수령
2.1
마이페이지 (대시보드)
/my

로그인 후 가장 먼저 보이는 회원 전용 메인 페이지입니다. 프로필 영역, 상태 카드(대기/승인/급여), 빠른 링크, 다가오는 근무 일정을 한눈에 확인할 수 있습니다.

humendhr.com/my
영역내용
프로필 영역프로필 사진, 이름, 환영 메시지, 비밀번호 수정/회원탈퇴 버튼
상태 카드 (3개)대기 중 지원 (노란), 승인된 지원 (녹색), 이번 달 급여 (회색)
바로가기 (4개)프로필 관리, 근무신청 조회, 근무내역, 급여신청
다가오는 근무D-day 배지로 임박한 근무 일정 표시
2.2
프로필 관리 (이력서)
/my/resume

회원 정보를 등록하고 수정하는 핵심 페이지입니다. 일자리에 지원하려면 반드시 프로필을 완성해야 합니다. 기본 정보, 본인인증, 경력 사항, 계좌 정보, 개인정보 동의 섹션으로 구성됩니다.

humendhr.com/my/resume
섹션필드
기본 정보이름(읽기전용), 전화번호(읽기전용), 생년월일, 성별, 이메일, 지역, 신장
본인인증주민번호 앞/뒷자리, NICE 인증 버튼
경력 사항경력 여부 토글, 상세 경력 텍스트
계좌 정보은행명 드롭다운(21개 은행), 예금주, 계좌번호
개인정보 동의개인정보처리방침 전문 + 동의 체크박스
2.3
근무신청 조회
/my/applications

내가 지원한 모든 일자리의 상태를 확인하는 페이지입니다. 요약 카드(대기/승인/거절)와 탭 메뉴로 필터링할 수 있습니다.

humendhr.com/my/applications
2.4
급여신청
/my/salary

계약서 서명 및 급여 신청을 처리하는 핵심 페이지입니다. 계좌 정보 확인, 미서명 근무 기록에 대한 계약 체결(전자서명), 서명 완료 후 급여 지급 대기 상태를 관리합니다.

humendhr.com/my/salary
기능설명
계약 체결서명 패드에 직접 서명 → 계약 확정
계약서 보기서명된 계약서 PDF 확인
급여명세서세전/공제/실수령액 상세 보기
직접 급여신청고객사 선택 후 수동 급여 신청
2.5
계약서 조회
/my/contracts

서명 완료된 모든 계약서를 조회하는 페이지입니다. 고객사명, 근무일, 실수령액, 상태를 확인할 수 있습니다.

humendhr.com/my/contracts
2.6
근무내역
/my/history

월별 근무 내역과 급여 요약을 확인하는 페이지입니다. 월간 통계(근무일수, 총 근무시간, 실수령 급여)와 상세 기록을 조회합니다.

humendhr.com/my/history
03 — Admin Pages

관리자 메뉴

관리자 계정으로 로그인해야 접근 가능합니다. 좌측 사이드바로 모든 관리 메뉴에 접근합니다.

관리자 로그인
→
고객사 등록
→
일자리 등록
→
지원 승인/거절
→
급여 승인
→
지급 확인
3.1
관리자 대시보드
/admin

전체 현황을 한눈에 파악하는 메인 페이지입니다. 5개 통계 카드, 지원 현황 파이 차트, 월별 급여 추이 바 차트, 미처리 지원 목록, 최근 활동 타임라인을 제공합니다.

humendhr.com/admin
통계 카드내용색상
확정 근무확정된 근무 기록 수파랑
미처리 지원승인 대기 중인 지원 수노랑
등록 회원전체 등록 회원 수보라
제휴 고객사등록된 고객사 수녹색
이번 달 급여당월 총 급여 지급액핑크
3.2
회원 관리
/admin/members

등록된 전체 회원을 조회하고 상세 정보를 확인합니다. 검색바로 이름/전화번호 검색, 이름 클릭 시 회원 상세 모달이 열립니다.

humendhr.com/admin/members
3.3
고객사 관리
/admin/clients

제휴 고객사(업체)를 등록, 수정, 삭제, 순서 변경하는 페이지입니다. 드래그로 표시 순서를 변경할 수 있습니다.

humendhr.com/admin/clients
3.4
일자리 관리
/admin/jobs

일자리(근무 슬롯)를 등록하고 관리합니다. 고객사별로 카드가 표시되며, 각 카드에서 날짜/시간/인원을 설정한 슬롯을 추가합니다.

humendhr.com/admin/jobs
3.5
지원 관리
/admin/applications

회원들의 일자리 지원을 승인/거절하는 페이지입니다. 대기/승인/거절/전체 탭으로 필터링하며, 승인/거절 버튼은 "대기" 탭에서만 표시됩니다.

humendhr.com/admin/applications
3.6
급여 관리
/admin/payroll

회원의 급여를 확인하고 처리합니다. 서명 완료 → 관리자 승인 → 급여 지급 확인 → 지급 완료 흐름으로 처리합니다.

humendhr.com/admin/payroll
3.7
급여지급 내역
/admin/payments

지급 완료된 급여 내역을 조회하는 페이지입니다. 월별 조회, 수령자, 금액, 지급 방법, 상태를 확인합니다.

humendhr.com/admin/payments
3.8
계약서 관리
/admin/contracts

모든 서명 완료된 계약서를 조회합니다. 회원 이름 클릭 시 계약서 보기 모달(서명 이미지 포함)이 열리며, PDF 다운로드도 가능합니다.

humendhr.com/admin/contracts
3.9
설정 (관리자 계정)
/admin/settings

관리자 계정을 추가하고 관리하는 페이지입니다. 관리자 ID, 이름, 비밀번호를 입력하여 새 관리자를 생성하거나 삭제할 수 있습니다.

humendhr.com/admin/settings
